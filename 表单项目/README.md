# 表单项目

用于腾讯文档智能表格页面的数据提取。脚本会在页面上挂载“提取我的排期”按钮，点击后展示当前登录名命中的行，并按排期过滤：

- 当前登录名：优先从右上角账号昵称识别；识别失败时可在弹窗内手动填写并保存。
- 人员匹配：优先匹配“参与测试人员 / 负责人 / 处理人 / 开发 / 测试 / 成员”等人员列。
- 排期过滤：使用“排期开始时间”和“排期结束时间”列；开始时间可以早于今天，但排期结束日期必须晚于今天。结束日期等于今天也会排除，例如今天是 14 号时，12 号到 18 号保留，结束到 14 号排除。
- 展示结果：弹窗内展示命中行，支持复制为表格文本或 JSON。

## 安装

1. 打开 Tampermonkey。
2. 新增脚本，粘贴 `表单数据提取.user.js` 的内容。
3. 保存后打开腾讯文档智能表格页面。
4. 刷新页面一次，等待表格加载完成后点击“提取我的排期”。

腾讯文档智能表格主体是 canvas 渲染，页面 DOM 不直接包含单元格文本。脚本需要从页面加载过程中的接口响应、本地缓存或 IndexedDB 里捕获表格数据，所以首次安装后已经打开的页面必须刷新一次。

## 调试

在页面控制台执行：

```js
localStorage.setItem('form-project-current-user-extractor:debug', '1')
```

刷新后可查看捕获日志。关闭调试：

```js
localStorage.removeItem('form-project-current-user-extractor:debug')
```
