- 更新编码。

## 2026-04-25

- 修复 macOS 上 `Alt + R` 快捷键失效问题，改用 `e.code === "KeyR"` 判断物理键。
- 仪表盘截图的剪贴板写入改用 `ClipboardItem(Promise)`，缓解 Safari 非用户手势拒绝。
- 剪贴板不可用时自动下载 PNG，作为 Safari / 隐私模式 / HTTP 场景的退化方案。

## 2026-04-23

- 新增统一设置面板：候选报障人、门户标题前缀、仪表盘 pageId / boardId / gadget IDs、共享组等均可在 UI 编辑，并持久化到 localStorage。
- 工具栏新增“设置”按钮，支持一键重置默认。

## 2026-04-22.2

- 修复仪表盘截图预览被底部按钮遮挡：内容区可滚动，预览图限制最大高度。
- 复制按钮（复制到剪贴板 / 复制文字）点击后不再自动关闭弹窗。
- 元信息补充 `@author`，描述加入“截图预览”。

## 2026-04-22

- 新增仪表盘截图功能：一键生成 4 宫格缺陷分布 SVG，导出 PNG 并复制到剪贴板。
- 截图后在弹窗内内嵌预览，支持点击放大在新标签打开。
- 报告弹窗支持合并多个测试执行、报障人候选勾选、测试小结输入。
- 已执行统计开关：可切换是否将阻塞 / 执行中计入已执行。
- 仪表盘配置：自动创建筛选器 + 更新门户标题 + 同步 4 个 gadget。
- Xray 报告列表页每行加入“生成报告”按钮，新标签自动生成。
- 支持 `Alt + R` 快捷键唤起生成报告。
